🎮 HOW TO PLAY

  KEYBOARD:
    Arrow Keys / A,D    →  Move left/right
    Space / W / ↑       →  Jump
    Shift / Z           →  Run fast
    X / F               →  Shoot fireballs (with 🌸)
    P                   →  Pause

  MOBILE (Touch):
    ◀ ▶                 →  Move left/right
    JUMP                →  Jump (hold for higher!)
    RUN                 →  Run fast
    🔥                  →  Shoot fireballs

🎯 GAMEPLAY TIPS

  JUMP TIMING
    • Can jump up to 0.2 seconds after walking off a platform (coyote time!)
    • Jump input is buffered for 12 frames (very forgiving)
    • Hold jump for max height, tap for short hop

  NEW: WALL SLIDING
    • Jump into a wall while falling
    • You'll slide down smoothly instead of bouncing
    • Use it to reach new platforms or find secret areas!

  STOMPING ENEMIES
    • Jump on enemy heads from above
    • Get bounced higher for each stomp
    • Chain multiple stomps = COMBO for more points!

  FIRE FLOWER
    • Shoot fireballs with X or F key
    • Much faster fire rate now (18 frame cooldown)
    • Great for breaking bricks and hitting distant enemies
