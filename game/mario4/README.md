## 🎮 How to Play

**Desktop:**
- Arrow keys / WASD = Move
- Space / W / ↑ = Jump
- Shift / Z = Run fast
- X / F = Shoot fireballs
- P = Pause

**Mobile:**
- Touch buttons are now HUGE (70px) and clear
- "JUMP" button on bottom-right
- "RUN" button next to JUMP
- Left/Right buttons on bottom-left
