# Node.js SDK — Getting Started

## WHY

Use this path if you want a fast, friendly first impression of the Node SDK.

## WHAT TO RUN

1. [../basic_usage.mjs](../basic_usage.mjs) — simplest useful chat flow
2. [../session_aware_support.mjs](../session_aware_support.mjs) — context-aware app pattern
3. [../parallel_research.mjs](../parallel_research.mjs) — fan-out and synthesis

## HOW

```bash
cd sdks/nodejs-native
npm install
ollama pull gemma4:latest
node examples/basic_usage.mjs
```

When you want something more demoable, continue with [../business/README.md](../business/README.md).
